## できること

A reproducible workspace for comparing model outputs and evaluation notes. プロジェクトの README にある手順で、環境に合わせて実行してください。

## はじめ方

認証情報や個人データはリポジトリに保存せず、環境変数または Git で無視されるローカル設定を使います。
